# Plot Baseline Intensity

Plot Baseline Intensity

## Usage

``` r
plot_baseline(x, ...)
```

## Arguments

- x:

  A [`dem`](https://corneliusfritz.github.io/redeem/reference/dem.md),
  [`rem`](https://corneliusfritz.github.io/redeem/reference/rem.md), or
  [`redeem_result`](https://corneliusfritz.github.io/redeem/reference/summary.redeem_result.md)
  object.

- ...:

  Additional arguments passed to \`graphics::plot\`.
